# gl.vertexAttribPointer() 的步进和偏移参数 

使用多个缓冲区对象向着色器传递多种数据，比较适合 *数据量不大* 的情况。

而当复杂的三维图形具有成千上万个顶点的时候，维护所有的顶点数据是困难的，因此 WebGL 允许我们把顶点的坐标和尺寸数据打包到同一个缓冲区对象中，然后通过某种机制分别访问缓冲区对象中不同种类的数据。

比如:
```javascript
var verticesSizes = new Float32Array([
    // 顶点的坐标和尺寸
    0.5, 0.5, 10.0,   // 第一个点
    -0.5, -0.5, 20.0, // 第二个点
    0.5, -0.5, 30.0   // 第三个点
]);
```
